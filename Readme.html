Sistema de Control de Acceso a Estudiantes
Proyecto Gestión De Ingreso

Objetivos:

General:
Establecer el concepto de un Sistema de Gestión de Ingreso, que pueda solucionar las problemáticas de la institución en cuanto al Registro de Ingreso y Salida de estudiantes, profesores, padres o acudientes.

Específicos:

  1. Recolectar información necesaria para la revisión de las problemáticas que presente la institución.
  2. Interpretar el estado actual de el sistema de ingreso de la institución.
  3. Crear una alternativa parcial de un sistema que pudiera mejorar la condición del sistema.

Planteamiento del Problema

  • El registro de estudiantes solo se hace para los que llegan tarde, los que llegan temprano no tienen ningún tipo de registro.

  • La ruta no entrega registro de los estudiantes que recoge o deja en la institución. Opcional (dígame si le redacto situación a esta problemática)

  • Los estudiantes de preescolar son los que más llegan tarde, pero ellos no reciben ningún tipo de retardo o llamado de atención.

  • Los retardos se registran de forma escrita en papel.

Justificación:

El problema que existe es que el ingreso se registra en papel. Esto es un problema porque no se tiene un control eficaz de esta situación, realizar llamados de atención por reincidir en estos comportamientos se pueden ver entorpecidos al no tener un sistema claro con información almacenada y organizada adecuadamente que este a la mano para efectuar el llamado de atención.

El problema no es grave, pero tener esta información organizada y disponible en cualquier momento significa de mucha utilidad.

Alcance:

Desarrollaremos un Software que permita gestionar la entrada y salida de estudiantes durante sus horas que efectúen de estudio.
