Sistema de Control de Acceso a Estudiantes
Proyecto Gestión De Ingreso

Objetivos:

General:
Establecer el concepto de un Sistema de Gestión de Ingreso, que pueda solucionar las problemáticas de la institución en cuanto al Registro de Ingreso y Salida de estudiantes, profesores, padres o acudientes.

Específicos:

  1. Recolectar información necesaria para la revisión de las problemáticas que presente la institución.
  2. Interpretar el estado actual de el sistema de ingreso de la institución.
  3. Crear una alternativa parcial de un sistema que pudiera mejorar la condición del sistema.
